# Niliko
Simple Electron.js app. <br>
2 Pages, Login and Profile

# Depends
- Node.js 14+ Version
- NPM 6+ Version
- Brains

# Install
```bash
$ cd niliko-main
$ npm i
```

# Run
```bash
$ npm i electron:serve
```
